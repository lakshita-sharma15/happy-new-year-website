Happy New Year Web Project
This is a simple Happy New Year greeting website created using HTML, CSS, and JavaScript. The project displays a festive New Year message with a clean and attractive design along with basic interactivity. It is a beginner-friendly front-end project that helps in understanding the fundamentals of web development and JavaScript behavior.
This project can be run by simply opening the `index.html` file in any web browser. It is ideal for students who are starting with web development and want to practice structuring, styling, and adding interactivity to a webpage.
